# Проект: Место

### Технологии

* Grid Layout
* Flexbox
* Адаптивная верстка
* Java Script

**Описание**

Проект о путешествиях Mesto. Адаптирован под распространённые разрешения экранов (1280px, 1024px, 768px, 320px).
Есть возможность редактирования информации профиля.
